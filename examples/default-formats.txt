# Local default formats list — used by prismaquant-llama pipeline runs
# when --formats is not passed on the CLI.
#
# Precedence: CLI --formats > this file > built-in PipelineConfig default.
# To revert to the built-in default, just delete this file.
#
# This list combines:
#   - mainline llama.cpp formats (works on any binary)
#   - frankenturbo2 IK-K extensions (works only on ik_llama / frankenturbo2 builds)
#
# Confirmed working on /path/to/llama-fork/build/bin/llama-quantize
# (run `prismaquant-llama discover <binary>` to verify your binary supports them).
#
# IQ2_K is intentionally NOT included — documented +30 PPL cliff vs F16
# across every model measured. IQ4_KT is excluded as slow (trellis quant).
