# To revert to built-in, just delete the copy.

# This example contains MAINLINE llama.cpp formats only — works on any
# binary including stock ggml-org/llama.cpp. If your binary is a fork

# append those format names below — run `prismaquant-llama discover
# <binary>` to see what your binary supports.
